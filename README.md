# Shanghai-Metro-Planner-EasyX
基于图结构的上海地铁换乘指南系统，支持地铁线路和站点的动态添加、地铁网络图形化显示、起终点路径查询以及乘车换乘路线展示。A graph-based Shanghai Metro transfer guide system that supports dynamic metro line and station management, visual metro network display, route planning, and transfer path visualization.

目前是一坨屎，如果你想使用的话，请下载EasyX，然后build然后运行。
